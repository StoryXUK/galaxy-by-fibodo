Key Functionality

Role-Based Access Control

Set standard roles (e.g. admin, manager, coach, finance) or create your own. Restrict access to features, locations, reports, and data fields so people only see what they need.

Granular Permissions

Define who can create, edit, cancel, refund, export, or delete. Apply rules per team, site, or brand so multi-site and franchise operations stay consistent and compliant.

Authentication & Sessions

Secure login flows with strong password policies, session timeouts, and device/session tracking to reduce the risk of shared or unattended accounts.

Payments & Checkout Security

Work with PCI-compliant payment gateways and strong customer authentication (SCA)-ready flows to help protect card data and reduce fraudulent transactions.

Data Protection & Privacy

Centralise client profiles, consents, and preferences with tools designed to support GDPR, including subject data requests, data minimisation, and clear access controls.

Audit Trails & Activity Logs

See who did what and when across key actions: bookings, membership changes, refunds, payouts, content edits, and permission updates.

Content & File Controls

Protect video libraries, documents, and resources with access rules linked to products, memberships, and roles, so only entitled users can view or download.

API & Integration Security

Use API keys and webhooks with scoped permissions so external tools (apps, kiosks, partners) can only access what they’re supposed to.

Environment & Infrastructure

Run on professionally managed infrastructure with monitoring, logging, and layered defences designed to reduce downtime and help mitigate common threats.

Backups & Resilience

Regular backups and disaster-recovery measures help protect against data loss and keep operations running if something goes wrong.

Compliance & Governance

Configurable policies, terms, and notices across brands and locations to support internal governance, regulator expectations, and client trust.
